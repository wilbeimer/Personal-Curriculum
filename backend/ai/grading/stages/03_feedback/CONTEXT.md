## Inputs
- assignment_title = {assignment_title}

## Process
Using the analysis and score from previous stages, write clear and constructive feedback for the student.
Feedback should be encouraging but honest, specific rather than generic, and actionable.
Do not repeat the score — focus on what the student did well and what they should improve.

## Output
Return as JSON:
{
    "feedback": "..."
}
